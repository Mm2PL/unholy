# Unholy
This project is an **experimental** Python to Javascript transpiler.

## Usage
Run the `unholy` package, `python -m unholy`.

```
usage: __main__.py [-h] [-o FILE] [-q] FILE

positional arguments:
  FILE

optional arguments:
  -h, --help            show this help message and exit
  -o FILE, --output FILE
  -q, --stfu
```
